# OSINT Bot 🤖

Telegram‑бот для сбора и анализа открытых данных (OSINT).  
Работает с GitHub API, RSS новостями и системой alert'ов.  

## 🚀 Возможности
- `/github username` — сохранить публичные данные GitHub‑профиля  
- `/news` — собрать свежие новости (RSS)  
- `/search keyword` — поиск по базе собранных данных  
- `/alert keyword` — добавить ключевое слово для мониторинга  
- `/listalerts` — показать список всех alert'ов  
- `/removealert keyword` — удалить alert  
- `/whois domain.com` — проверить домен через WHOIS API  
- `/ipinfo IP` — информация по IP (гео, провайдер)  
- `/stats` — статистика базы (профили, alerts)  
- `/randomfact` — случайный факт для фана  
- `/gaz` — мемная команда «Газ в пол!»  
- `/meme` — случайный мемный ответ  
- `/help` — показать меню команд
